Welcome to EB4j(forked) project.

EB4J is a 100% pure java library to access EPWING electronic dictionaries. It is derivered from EB library in C. [http://www.sra.co.jp/people/m-kasahr/eb/index.html] The original project has been developed by Mr. Hisaya FUKUMOTO at OSDN [http://eb4j.osdn.jp/]

License

EB4J is distributed on LGPL2.1.

Difference from the original

  • It is located in package 'io.github.eb4j' than original 'fuku.eb4j'.
  • It is targeted and compiled on Java8 than Java6 for original.
  • It is updated with some bug fixes.
  • It will be released on jceter repository.
  • It uses Gradle for build system, against original uses Maven.
